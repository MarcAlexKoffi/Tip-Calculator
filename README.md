# Tip Calculator App
Calculateur de pourboire est une application Android simple et intuitive développée en Kotlin avec Jetpack Compose. 
Elle permet de calculer rapidement un pourboire en euros à partir du montant d'une addition, avec la possibilité d’arrondir automatiquement le résultat.

# Fonctionnement du projet
L’utilisateur saisit le montant de l’addition et le pourcentage de pourboire souhaité.
L’application calcule automatiquement le montant du pourboire correspondant, l’affiche au format euro (€), et propose une option pour arrondir le résultat à l’entier supérieur.
L’interface repose sur Jetpack Compose, avec des champs de saisie stylisés, des icônes Material et un switch interactif.
Tous les calculs sont effectués en temps réel, et le projet inclut des tests automatisés pour garantir la fiabilité du comportement attendu.

# Technologies utilisées
- Kotlin
- Jetpack Compose
- Android Studio version Meerkat Feature Drop 2024.3.2 Patch 1
